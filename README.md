# 🪁 Kite Fight Dice Game

> 🏆 Built for the **Build Games with Amazon Q CLI** Challenge by AWS

A Java-based terminal dice game inspired by the classic South Asian sport of kite fighting. Two players compete by rolling dice each round — the player with the higher roll cuts the opponent's kite string. The first player to win the required rounds claims victory and becomes the **Kite Champion**!

This game was built entirely using **Amazon Q CLI** through natural language prompts, making it an exploration of AI-assisted game development.

---

## ✨ Features

- **2-Player Dice Combat** — Both players roll dice each round; highest roll wins the round
- **Kite Fight Theme** — South Asian kite fighting culture brought to life in a terminal game
- **Round-Based Gameplay** — First to reach the winning round count takes the match
- **Turn-by-Turn Narration** — Console output narrates each dice roll and round result
- **Clean Java OOP Design** — Modular, well-structured Java code with game logic separated from I/O
- **AI-Generated with Amazon Q CLI** — Built using natural language prompts via AWS's AI-powered developer tool

## 🚀 Getting Started

### Prerequisites
- Java JDK 8 or above installed

### 1. Clone the Repository

```bash
git clone https://github.com/malikKanishka/Kite_Fight_Dice_Game.git
cd Kite_Fight_Dice_Game/Kite_Fight_V1.0.0
```

### 2. Compile

```bash
javac *.java
```

### 3. Run

```bash
java KiteFightGame
```

## 🎮 How to Play

```
1. Two players take turns rolling a dice
2. The player with the higher dice roll wins the round
3. A winning roll = your kite cuts the opponent's string
4. First player to win the required number of rounds wins the match
5. The winner is crowned Kite Champion! 🏆
```

### Example Round
```
--- Round 1 ---
Player 1 rolls: 5 🎲
Player 2 rolls: 3 🎲
Player 1 wins this round! ✂️ String cut!

--- Round 2 ---
Player 1 rolls: 2 🎲
Player 2 rolls: 6 🎲
Player 2 wins this round! ✂️ String cut!
```

## 🤖 Built with Amazon Q CLI

This game was created as part of the **Build Games with Amazon Q CLI** challenge by AWS. The entire game was developed by describing game mechanics and rules in **natural language** to Amazon Q CLI, which generated the Java code.

This project demonstrates how AI-powered developer tools can accelerate game development — from concept to playable game — through conversational prompts.
